# Simplified-Trading-Bot-Binance-Futures-Testnet-

It is a binance future testnet project which place order for BUY and SELL.
1.Write a code for Registration user in which using django built-in user model and generate acess and refresh token
2. Then login a user with their credentials.
3.Take a input from user for place a order and check the inputs are valide(like price >0,quantitity>0) and store in Orders model
4.then call a Binance api for placing a order and if place then it's details save in Order's model

i have run the api on postman application where i check the login ,register , place order work properly
